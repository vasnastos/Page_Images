Εγκατάσταση jdk

  • Τρέχουμε το .msi αρχείο

  • Πατάμε το κουμπί next

  • Τσεκάρουμε τημ επιλογή I accept the terms in the license Aggriment και πατάμε next

  • Πατάμε το κουμπί next

  • Πατάμε το κουμπί Install


  • Ελέγχουμε αν το development kit έχει εγκατασταθεί στον υπολογιστή μας,εκτελώντας στην γραμμή εντολών την εντολή java -version
  • Ελέγχουμε αν ο compliler για την γλώσσα java έχει εγκατασταθεί σωστά,εκτελώντας στην γραμμή εντολώ την εντολή javac -version


    • Αν εμφανίζονται τα παρακάτω μυνήματα έχετε ολοκληρώσει την Εγκατάσταση του development kit και του compiler της javaκαι μπορείτε να γράψεται και να μεταγλωττίσεται ένα πρόγραμμα σε java.
